# Zombie Basic Shield 
Raspberry PI compatible shield for Zombie Blackheart Daemon

Features:
- 8 up to 24v 10-bit channels
- 16 up to 24v digital channels
- 4 ground controlled channels
- 2 high speed optical isolated channels
- Real Time CLock
- OneWire interface
